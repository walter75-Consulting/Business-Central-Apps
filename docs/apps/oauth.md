## 📋 Overview

This extension provides a **comprehensive OAuth 2.0 authentication framework** for Business Central, enabling secure integration with external APIs and services. It handles token management, refresh flows, and credential storage following OAuth 2.0 best practices.

<div class="feature-grid">
  <div class="feature-card">
    <h3>🔒 Secure by Design</h3>
    <p>Encrypted credential storage and token management following industry security standards</p>
  </div>
  <div class="feature-card">
    <h3>🔄 Automatic Refresh</h3>
    <p>Intelligent token lifecycle management with automatic expiry handling and renewal</p>
  </div>
  <div class="feature-card">
    <h3>🌐 Multi-Provider</h3>
    <p>Configure and manage multiple OAuth applications from different providers</p>
  </div>
  <div class="feature-card">
    <h3>⚡ Integration Ready</h3>
    <p>Built for seamless integration with external REST APIs and web services</p>
  </div>
</div>

## ✨ Key Features

<div class="info-box">
  <ul>
    <li><strong>OAuth 2.0 Token Management</strong>: Secure token storage and automatic refresh</li>
    <li><strong>Multiple Provider Support</strong>: Configure multiple OAuth applications simultaneously</li>
    <li><strong>Token Lifecycle</strong>: Automatic token expiry handling and renewal</li>
    <li><strong>Secure Credential Storage</strong>: Encrypted storage of client secrets and tokens</li>
    <li><strong>API Integration Ready</strong>: Easy integration with external REST APIs</li>
  </ul>
</div>

## 📦 Installation

<div class="step-container">
  <div class="step">
    <div class="step-number">1</div>
    <div class="step-content">
      <h4>Download Extension</h4>
      <p>Download the latest <code>.app</code> file from <a href="https://github.com/walter75-Consulting/Business-Central-Apps/releases">GitHub Releases</a></p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">2</div>
    <div class="step-content">
      <h4>Install in Business Central</h4>
      <p>Open <strong>Extension Management</strong> in Business Central and upload the app file</p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">3</div>
    <div class="step-content">
      <h4>Configure OAuth</h4>
      <p>Open <strong>OAuth Applications</strong> page and configure your OAuth providers</p>
    </div>
  </div>
</div>

## 🛠️ Configuration

<div class="info-box info-box-warning">
  <h4>Quick Setup Guide</h4>
  <ol>
    <li>Navigate to <strong>OAuth Applications</strong> page in Business Central</li>
    <li>Click <strong>New</strong> to create a new OAuth application entry</li>
    <li>Configure the following fields:
      <ul>
        <li>Client ID (from your OAuth provider)</li>
        <li>Client Secret (securely stored)</li>
        <li>Authorization URL</li>
        <li>Token URL</li>
      </ul>
    </li>
    <li>Save and test the connection</li>
  </ol>
</div>
