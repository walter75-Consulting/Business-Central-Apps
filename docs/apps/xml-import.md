# 📄 walter75 - XML Import

> XML data import functionality for external system integration

## 📋 Overview

XML Import provides flexible XML data import capabilities for Business Central. Import data from external systems, EDI partners, or legacy applications using configurable XML mappings.

## ✨ Features

- **Flexible XML Parsing**: Handle various XML formats and structures
- **Field Mapping**: Configure mappings between XML elements and BC fields
- **Validation Rules**: Validate imported data before processing
- **Error Handling**: Detailed error logs and rollback support
- **Batch Processing**: Import large XML files efficiently
- **Transformation Support**: Transform XML data during import

## 🔢 Object ID Range

**ID Range**: 60500-60799

## 📦 Installation

1. Download the latest `.app` file from [Releases](../../releases)
2. Install via Business Central Extension Management
3. Configure XML import definitions

## 🔗 Dependencies

No dependencies - this is a standalone extension.

## 🛠️ Configuration

After installation:
1. Create XML import definition
2. Map XML elements to Business Central tables/fields
3. Set up validation rules
4. Test with sample XML file
