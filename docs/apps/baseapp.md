## 📋 Overview

BaseApp Basic serves as the foundation layer for the walter75 Business Central app ecosystem. It provides shared functionality, common utilities, and base configurations used by other extensions in the suite.

<div class="feature-grid">
  <div class="feature-card">
    <h3>🛠️ Shared Utilities</h3>
    <p>Common functions used across multiple apps</p>
  </div>
  <div class="feature-card">
    <h3>⚙️ Base Configurations</h3>
    <p>Centralized setup and configuration management</p>
  </div>
  <div class="feature-card">
    <h3>🏗️ Foundation Layer</h3>
    <p>Core functionality for dependent apps</p>
  </div>
  <div class="feature-card">
    <h3>🔌 Extensibility</h3>
    <p>Enables consistent extension patterns</p>
  </div>
</div>

## 🔢 Technical Details

<div class="info-box info-box-primary">
  <p><strong>Object ID Range:</strong> 80000-80099</p>
  <p><strong>Dependencies:</strong> None (base-level extension)</p>
  <p><strong>Target Platform:</strong> Business Central Cloud 27.0+</p>

## 📦 Installation

<div class="step-container">
  <div class="step">
    <div class="step-number">1</div>
    <div class="step-content">
      <h4>Download Extension</h4>
      <p>Download the latest <code>.app</code> file from <a href="https://github.com/walter75-Consulting/Business-Central-Apps/releases">Releases</a></p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">2</div>
    <div class="step-content">
      <h4>Install First</h4>
      <p>Install via <strong>Extension Management</strong> - this must be installed <strong>before</strong> other walter75 extensions</p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">3</div>
    <div class="step-content">
      <h4>Verify Installation</h4>
      <p>Check that the extension is active and ready for dependent apps</p>
    </div>
  </div>
</div>

<div class="info-box info-box-warning">
  <p>⚠️ <strong>Important:</strong> Other walter75 extensions depend on this app. Install BaseApp Basic first before installing dependent extensions.</p>
</div>
