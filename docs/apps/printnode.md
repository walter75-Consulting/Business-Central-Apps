## 📋 Overview

PrintNode integration enables direct printing to network printers, thermal label printers, and local printers from Business Central. Ideal for automated label printing in warehouse and logistics scenarios without user interaction.

<div class="feature-grid">
  <div class="feature-card">
    <h3>☁️ Cloud Printing</h3>
    <p>Print to any printer via PrintNode service</p>
  </div>
  <div class="feature-card">
    <h3>🏷️ Thermal Printers</h3>
    <p>Optimized for Zebra, Dymo, and other label printers</p>
  </div>
  <div class="feature-card">
    <h3>🔇 Silent Printing</h3>
    <p>Background printing without user dialogs</p>
  </div>
  <div class="feature-card">
    <h3>🎯 Multi-Printer</h3>
    <p>Route document types to specific printers</p>
  </div>
  <div class="feature-card">
    <h3>📋 Label Templates</h3>
    <p>Support for ZPL and other label formats</p>
  </div>
  <div class="feature-card">
    <h3>📊 Queue Management</h3>
    <p>Track print jobs and handle errors</p>
  </div>
</div>

## 📦 Installation

<div class="step-container">
  <div class="step">
    <div class="step-number">1</div>
    <div class="step-content">
      <h4>Sign up for PrintNode</h4>
      <p>Create account at <a href="https://www.printnode.com">printnode.com</a></p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">2</div>
    <div class="step-content">
      <h4>Install PrintNode Client</h4>
      <p>Install PrintNode client on your print server</p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">3</div>
    <div class="step-content">
      <h4>Install Extension</h4>
      <p>Download and install the <code>.app</code> file via Extension Management</p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">4</div>
    <div class="step-content">
      <h4>Configure API Key</h4>
      <p>Enter your PrintNode API key in <strong>PrintNode Setup</strong></p>
    </div>
  </div>
  
  <div class="step">
    <div class="step-number">5</div>
    <div class="step-content">
      <h4>Map Printers</h4>
      <p>Map printers to document types and test printing</p>
    </div>
  </div>
</div>
